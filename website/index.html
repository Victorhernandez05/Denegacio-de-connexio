Domina la teva Xarxa

NetGuardian és el bot de Telegram més potent per gestionar la connectivitat local, escanejar dispositius i protegir el teu entorn.

Començar ara

Funcionalitats Premium

Eines de nivell professional en una interfície senzilla.

🛡️

Control Local

Bloqueja tot el trànsit d'internet de l'ordinador local mantenint la connexió amb Telegram via iptables.

🔫

ARP Spoofing

Desconnecta qualsevol dispositiu de la xarxa local (Kick) de forma remota amb un simple missatge.

🔍

Escàner de Xarxa

Llista tots els dispositius actius, les seves IPs i adreces MAC en temps real.

Interfície Telegram

User: /dispositius
Bot: 🔍 Escanejant la xarxa...
Bot: 📡 Dispositius a la xarxa:
`192.168.1.50` — `00:0c:29:43:c2:d5` 🟢 en línia
User: /kick 192.168.1.50
Bot: ✅ Dispositiu 192.168.1.50 kickejat correctament.

Instal·lació Ràpida

1. Clona el Repo

git clone https://github.com/Victorhernandez05/Denegacio-de-connexio.git

2. Dependències

pip install -r requirements.txt

3. Executa

sudo python3 bot.py
